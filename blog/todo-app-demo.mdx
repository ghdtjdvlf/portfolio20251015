# Todo App Demo

TanStack Query와 Jotai를 활용한 모던 할 일 관리 애플리케이션입니다.

## 🚀 라이브 데모

👉 [**Todo App Demo 바로가기**](https://sp-timer.web.app)

## 🎯 주요 기능

- ✅ 할 일 추가/삭제/완료 처리
- ✅ Optimistic Updates로 즉각적 UI 반응
- ✅ 필터링 (전체/진행중/완료)
- ✅ 실시간 검색
- ✅ LocalStorage 자동 저장
- ✅ TanStack Query 자동 캐싱 (5분)

## 🛠 기술 스택

### Frontend
- **React** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구

### 상태 관리
- **TanStack Query** - 서버 상태 관리 및 캐싱
- **Jotai** - 전역 상태 관리

### 스타일링
- **Emotion** - CSS-in-JS

## 📦 프로젝트 구성

이 프로젝트는 다음과 같은 구조로 구성되어 있습니다:

1. **Todo 앱** - 기본 할 일 관리
2. **Chatbot** - AI 챗봇 (Gemini API)
3. **Movies** - 영화 검색
4. **Pokemon** - 포켓몬 도감
5. **Kakao** - 카카오 API 연동
