## 기술 스택

- HTML5 / CSS3
- JavaScript
- Responsive Design
- 웹 접근성 (WCAG 2.1 준수)

## 협업 프로세스

### 역할 분담

- **기획 및 디자인**: 사용자 리서치, 와이어프레임 제작
- **프론트엔드 개발**: HTML/CSS/JS 구현, 반응형 레이아웃
- **테스트 및 QA**: 크로스 브라우징 테스트, 접근성 검증

## 프로젝트를 통해 배운 점

### 협업의 중요성

- **Git 협업**: 브랜치 전략과 코드 리뷰 프로세스
- **커뮤니케이션**: 정기 미팅과 진행상황 공유
- **역할 분담**: 효율적인 업무 분배와 일정 관리

### 기술적 성장

- **접근성 구현**: ARIA 속성과 시맨틱 HTML 활용
- **반응형 디자인**: 다양한 디바이스 대응 전략
- **크로스 브라우징**: 브라우저 호환성 처리 경험
