## 프로젝트 개요
이 웹사이트는 **퍼블리셔이자 기획자로서의 기술 역량과 작업물**을 한곳에 모아 정리한 개인 포트폴리오입니다.  
시각적 정갈함, 모바일 대응성, 정보 구조의 명확성을 목표로 제작했습니다.

### 주요 구성
- **히어로 섹션:** 이름, 직무, 소개 문구 및 프로필 이미지  
- **Skills 섹션:** 퍼블리셔 중심의 기술 스택을 카드 형태로 정리  
- **포트폴리오 섹션:** 주요 프로젝트(쇼핑몰·랜딩·기획 프로젝트 등) 썸네일형 전시  
- **푸터:** 기술 키워드 라인업 및 이메일 / SNS 링크 제공  

### 주요 기능
- **반응형 레이아웃** — PC, 태블릿, 모바일 환경에 최적화  
- **다크모드 베이스** — 블랙 톤 기반의 전문적 디자인  
- **카드형 UI 구성** — 기술 및 프로젝트를 시각적으로 정리  
- **외부 링크 연동** — 각 프로젝트 클릭 시 실서비스 페이지로 이동  
- **모듈형 관리 구조** — 새 프로젝트 추가 시 코드 수정 최소화  

### 기술 스택
**Frontend**
- HTML5 / CSS3 / JavaScript  
- Bootstrap5 기반 반응형 레이아웃  
- GSAP, Swiper 등 시각 효과 적용  

**Design & Tools**
- Figma / Photoshop  
- GitHub Pages 및 dothome 호스팅  
- Notion을 통한 콘텐츠 관리  

## 핵심 구현 사항

### 1. 반응형 레이아웃 설계
모든 섹션을 Bootstrap 5의 그리드 시스템 기반으로 구성하여  
해상도별(576px / 768px / 992px / 1200px)에서 자동으로 재배치되도록 설계했습니다.
```css
@media (max-width: 992px) {
  .portfolio-card { flex: 0 0 48%; }
}
@media (max-width: 576px) {
  .portfolio-card { flex: 0 0 100%; }
}
