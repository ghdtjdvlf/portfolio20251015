## 프로젝트 개요

그누보드 CMS 기반의 성형외과 웹사이트로, 병원 소개, 시술 정보, 예약 시스템, 게시판 등 병원 웹사이트에 필요한 핵심 기능들을 구현했습니다.

### 주요 기능

- **병원 소개**: 의료진 프로필, 시설 안내, 진료 시간
- **시술 정보**: 카테고리별 시술 상세 정보 제공
- **온라인 예약**: 실시간 예약 시스템
- **게시판**: 공지사항, 이벤트, 전후 사진, 후기
- **관리자 페이지**: 그누보드 관리 기능 활용

## 기술 스택

- GnuBoard 5
- PHP
- MySQL
- HTML5 / CSS3
- JavaScript / jQuery
- Responsive Web Design

```php
// 예약 폼 처리
if ($_POST['mode'] == 'reservation') {
    $name = clean_xss_tags($_POST['name']);

    $date = clean_xss_tags($_POST['date']);
    $time = clean_xss_tags($_POST['time']);
    $consultation = clean_xss_tags($_POST['consultation']);

                name = '$name',
                phone = '$phone',
                reservation_date = '$date',
                reservation_time = '$time',
                consultation_type = '$consultation',

    sql_query($sql);
}
```
