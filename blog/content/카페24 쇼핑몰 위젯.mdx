## 프로젝트 개요
기능 추가 시 각 쇼핑몰별로 FTP를 통해 HTML, CSS, JS를 개별 수정해야 했으나,
Firebase 기반의 중앙 스크립트 배포 시스템을 구축함으로써 유지보수 효율성을 대폭 개선했습니다.

### 주요 기능
- 브랜드별 색상·테마 자동 반영  
- PC/Mobile 디바이스별 UI 전환  
- JavaScript 단일 삽입 구조 (`<script>` 한 줄로 로드)  
- 코드 보기·시뮬레이터·버전 캐시 무효화 기능  
- Firebase Hosting 기반 배포 및 실시간 반영  

### 기술 스택
**Frontend**
- HTML5 / CSS3 / JavaScript(ES6)
- 반응형 구조 및 브랜드 컬러 매핑

**Hosting & Tools**
- Firebase Hosting (`sp-cafe24api.web.app`)
- 자동 버전 갱신(`?v=timestamp` 방식)
- Script 모듈: banner.js, timer.js, ship_timer.js, bnf_bar.js

## 핵심 구현 사항
모든 위젯을 section 단위로 분리해 테스트할 수 있도록 설계했으며, 각 브랜드별 테마 색상과 디바이스 설정에 따라 동적으로 스크립트를 불러오도록 구현했습니다.

```html
<div class="section">
  <div class="section-title">
    <span class="badge">timer.js</span>카운트다운 타이머
  </div>
  <div class="demo-box">
    <script src="https://sp-cafe24api.web.app/timer.js" data-brand="udit" data-device="pc"></script>
  </div>
</div>
